Quality: About This Data Set

As mentioned in the about page, the dataset includes battery and location data collected as a part of a study by Carnegie Mellon University using AWARE framework software.

Overall, the dataset is coherent and correct, with a little incomplete. For detailed explaination please see the following.

Is the data complete?

The data was incomplete because it can not collect data when phone or Aware App gets turned off. As you can see in following histagram demonstrating the common values in device usage dataset, out of 2286 data points, there are 1144 of them have zero device on time. This affects the my analysis from home page because this would introduce bias when analyzing phone usage of this dataset.

Is the data coherent?

Overall, this data is generally coherent. As shown in my analysis in home page, the scatter plots and quantile plot both proofed my assumption about the relationship between phone usage and battery level. Please see detailed analysis in home page.

Is the data correct?

This data is generally correct. As I stated in home page, the reason for not using my phone more often when it's fully charged might because of my activity pattern. So I counted the number of phone usage for each hour in these 23 days. For demonstration, I ploted counted phone usage for only Monday as following. It seems that the number of phone usage for some day of the week is zero from 8 am to 12 pm. This is usally my class period, so I tend to not use my phone during those time.
