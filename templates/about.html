Accountability: About This Data Set

This dataset contains my phone usage & battery level data between 1/28/17 and 2/19/17 stored in Google Cloud SQL. There are two relevant tables for this Byte:

  • plugin_device_usage: The table that contains Google activity data. It includes columns that represent the timestamp, elapsed device on time(ms), elapsed device off time(ms).
  • battery: The table that contains location data. It includes columns that represent the timestamp, battery status, batter level, battery scale, battery voltage, battery temperature, battery adaptor, battery health, battery technology.

Where did this data come from?

The dataset comes from a study of my battery & phone usage pattern by CMU Ubicomp Lab. The research uses AWARE framework, which stores various phone sensor data and, most importantly to this Byte, phone location and activity data. The location and activity data is sourced directly from relevant Google services. Data about me is periodically sent to an AWARE server that Ubicomp Lab manages, and the dataset for this Byte is a dump of my first 23 days' data in .sql format.

Who has access to it?

The data is part of the research data released to me. And, the two datasets included in this byte (device usage and battery data) are only accessible to Google, Ubicomp Lab, and me.
